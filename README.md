# 🚀 Запуск Own Your Finance на GitHub Pages

## Что внутри папки

```
oyf-deploy/
├── index.html      ← Главная страница (загружает React, Recharts, Babel)
├── app.jsx         ← Код приложения
├── manifest.json   ← PWA-манифест (иконка, имя, fullscreen)
├── sw.js           ← Service Worker (оффлайн-кеширование)
├── icon-192.png    ← Иконка 192×192
└── icon-512.png    ← Иконка 512×512
```

---

## Пошаговая инструкция (5 минут)

### 1. Создайте репозиторий на GitHub

1. Откройте **github.com** → войдите в аккаунт
2. Нажмите **+** (вверху справа) → **New repository**
3. Название: `own-your-finance` (или любое)
4. Оставьте **Public**
5. Нажмите **Create repository**

### 2. Загрузите файлы

1. На странице нового репозитория нажмите **uploading an existing file**
   (или **Add file → Upload files**)
2. Перетащите ВСЕ 6 файлов из папки `oyf-deploy/`:
   - `index.html`
   - `app.jsx`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Внизу нажмите **Commit changes**

### 3. Включите GitHub Pages

1. Перейдите в **Settings** (вкладка в репозитории)
2. В боковом меню: **Pages**
3. В разделе **Source** выберите:
   - Branch: **main**
   - Folder: **/ (root)**
4. Нажмите **Save**
5. Подождите 1–2 минуты

### 4. Откройте на телефоне

1. GitHub покажет ссылку вида:
   ```
   https://ваш-логин.github.io/own-your-finance/
   ```
2. Откройте эту ссылку **в Safari на iPhone**
3. Нажмите кнопку **«Поделиться»** (квадрат со стрелкой вверх)
4. Выберите **«На экран Домой»**
5. Нажмите **«Добавить»**

### ✅ Готово!

Приложение появится на домашнем экране как обычное приложение:
- Полноэкранный режим (без адресной строки Safari)
- Иконка на домашнем экране
- Работает оффлайн (после первой загрузки)
- Данные хранятся в localStorage на устройстве

## Обновление приложения

Если нужно обновить код:
1. В репозитории откройте файл `app.jsx`
2. Нажмите карандаш (Edit)
3. Замените содержимое
4. Нажмите **Commit changes**
5. Через 1–2 минуты сайт обновится

Если на телефоне не обновляется — откройте ссылку в Safari и обновите страницу, затем заново добавьте на домашний экран.

## Подключение Google Sheets (опционально)

1. Создайте Google Таблицу
2. Расширения → Apps Script
3. Вставьте код из файла `FinTrack-AppsScript.js` (скачан ранее)
4. Развернуть → Новое развёртывание → Веб-приложение → Все
5. Скопируйте URL
6. В приложении: Settings → вставьте URL таблицы и вебхука
7. Нажмите «Sync»
